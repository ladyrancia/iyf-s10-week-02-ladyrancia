# 🌐 My Portfolio Website

## 📌 Project Overview

This is my personal portfolio website built using **HTML and CSS**.
The goal of this project was to practice **responsive design, layout techniques, and styling** while creating a clean and modern user interface.

---

## 🎯 Features

* Responsive design (mobile, tablet, desktop)
* Navigation bar using Flexbox
* Project section using CSS Grid
* Styled contact form
* Hover effects and smooth transitions
* Multiple pages (Home, Projects, Contact)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
